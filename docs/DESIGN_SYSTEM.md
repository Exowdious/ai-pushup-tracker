## Accessibility

- High contrast (black on white/colors)
- Clear focus states
- Keyboard navigation support
- Screen reader friendly labels
- Touch-friendly button sizes (min 44x44px)

## Design Principles

1. **BOLD** - Everything is chunky and in-your-face
2. **RAW** - No gradients, no blur, sharp edges
3. **BRUTALIST** - Function over decoration
4. **COLORFUL** - Vibrant accent colors
5. **PLAYFUL** - Emojis and uppercase text
6. **HONEST** - No hiding, everything visible
