## ğŸ›‘ Stopping Servers

### Method 1: Keyboard (In Terminal)
```
Press: Ctrl + C

Output:
^C
KeyboardInterrupt
Shutting down gracefully...
```

### Method 2: Close Window
```
Click [X] on terminal window
Servers stop automatically
```

### Method 3: Kill Process (macOS/Linux)
```bash
# Find process
ps aux | grep "python backend.py"
ps aux | grep "vite"

# Kill by PID
kill -9 <PID>

# Or kill all
pkill -f "python backend.py"
pkill -f "vite"
```

### Method 4: Kill Process (Windows)
```cmd
REM Find process
tasklist | findstr python
tasklist | findstr node

REM Kill by name
taskkill /F /IM python.exe
taskkill /F /IM node.exe
```
